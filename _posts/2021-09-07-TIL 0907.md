### React & TypeScript

- 상수로 고정값이면 as const를 붙여준다.

    ```jsx
    const scores = {
    	a: 1,
    	b: 0,
    	c: -1
    } as const;
    ```

- typeof를 붙이면 정해놓은 타입의 값을 불러올 수 있다.

    ```jsx
    const testValue = {
    		a: 'abc',
    		b: 'def',
    		c: 'ghi',
    	} as const;

    const value = {
    		a: 1,
    		b: 2,
    		c: 3
    	} as const;

    type Test = typeof testValue[keyof typeof testValue];
    ```
